## Java Agent
| Parameter            | Description                                                                                                                       | Required | Example / Format                                                                                              |
|:---------------------|:----------------------------------------------------------------------------------------------------------------------------------|:---------|:--------------------------------------------------------------------------------------------------------------|
| agentpath            | the file stores the agent                                                                                                         | yes      | a relative or absolute path, for example: mingwX64-0.7.0-20/drill_agent.dll                                   |
| agentId              | name of agent                                                                                                                     | yes      | Petclinic                                                                                                     |
| adminAddress         | admin API URL                                                                                                                     | yes      | localhost:8090                                                                                                |
| drillInstallationDir | the folder stores agent files                                                                                                     | yes      | a relative or absolute path, for example:  mingwX64-0.7.0-20/                                                 |
| buildVersion         |                                                                                                                                   | ?        | 0.1.0                                                                                                         |
| instanceId           |                                                                                                                                   | no       |                                                                                                               |
| groupId              | this parameter groups your agents in one section. It is helpful if your app is based on a microservices arch                      | no       | petclinic-micro                                                                                               |
| logLevel             | logging agent work                                                                                                                | ?        | TRACE\|DEBUG\|INFO\|...                                                                                       |
| logFile              |                                                                                                                                   | no       | a relative or absolute path, for example: "E:\projects\project\spring-petclinic run with agent\drill-log.log" |
| isWebApp             |                                                                                                                                   | no       |                                                                                                               |
| isTlsApp             |                                                                                                                                   | no       |                                                                                                               |

| webAppNames          | the list of TODO                                                                                                                  | no       |                                                                                                               |

### System Variables
|                    Parameter                    |                       Description                      | Required |                         Example / Format                         |
|:----------------------------------------------- |:------------------------------------------------------ |:-------- |:---------------------------------------------------------------- |
| DRILL_HTTP_HOOK_ENABLED drill.http.hook.enabled | It can be true/false. By default = true.               | no       |                                                                  |
| DRILL_AGENT_CONFIG_PATH                         | The file stores the config file                        | no       | a relative or absolute path, for example:  configFile.properties |

### Example of How to Pass Parameters
You can list params by properties file:
```java
# config file:
drillInstallationDir=myVersion/
adminAddress=localhost:8090
agentId=Petclinic
buildVersion=0.1.0
logLevel=TRACE
```
And point to the config file in java args:
```java
java -agentpath:%DISTR%/drill_agent.dll=configPath=%DISTR%/configFile.properties -jar spring-petclinic-0.1.0.BUILD-SNAPSHOT.jar
```
Or use the system variable **DRILL_AGENT_CONFIG_PATH**.

Try Drill4J Demo Examples

We have prepared a demo environment for different agent types:

Java+JS App

Clone the following repository: Realworld Java and JS Coverage

  1. Start docker containers:
./demo-up.sh

  1. Go to http://localhost:8080and wait for application up.

  2. Go to Drill Admin http://localhost:8091 and register agents.

    Don't forget to set target host "http://localhost:8080/" for frontend agent.

  3. Run tests:

    ./gradlew clean :build1:test

    See results in the Drill Admin and finish scopes.

  4. Deploy the second build of the app:

    ./demo-deploy-build2.sh

    See new build notifications and diffs in the Drill Admin.

  5. Run tests:

    ./gradlew clean :build2:test

    See results in the Drill Admin and finish scopes.

  6. Deploy the third build of the app:

    ./demo-deploy-build3.sh

    See new build notifications and diffs in the Drill Admin.

  7. Run tests:

    ./gradlew clean :build3:test

    See results in the Drill Admin and finish scopes.

  8. Use browser extension for manual testing: Chrome Browser Extension Installation

    Also, you can down demo setup:

    ./demo-down.sh
Java App

  1. You need to download a special docker-compose-file: Docker-compose File

  2. Start Drill Admin:

    docker-compose up -d drill-admin admin-ui agent-files

  3. Start Example App with Drill Agent:

    docker-compose up -d example-app

  4. Use browser extension for manual testing: Chrome Browser Extension Installation

    You can find Drill Admin on http://localhost:8091

    And Application UI is located here: http://localhost:8087

    Application is based on Spring Petclinic

JS App

Clone the example repository:

git clone https://github.com/Drill4J/todomvc

Run Drill4J services:

  1. Run:

    docker-compose -f docker-compose.services.yml up -d

  2. Run docker ps and check that all services have started: drill-admin, admin-ui, jsagent and mongo.

  3. Open Drill Admin UI at http://localhost:9091

Run sample application:

  1. Run:

    docker-compose up -d

  2. Wait for "todomvc-typescript-angular" agent to appear in Admin UI at http://localhost:9091

  3. Press Register button.

  4. Click through Step 1 and Step 2, but make sure to enable Test2Code plugin at Step 3.

  5. Open sample application at http://localhost:8080

  6. Use browser extension for manual testing: Chrome Browser Extension Installation

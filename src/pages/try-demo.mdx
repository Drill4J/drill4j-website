We have prepared a demo environment for different agent types:
<!-- JAVA+JS BLOCK -->

<ExpandCollapse Icon={<JavaJSLogo />} title="Java+JS App">

1. Start docker containers:

```java
./demo-up.sh
```

<p>
    2. Go to <a href='http://localhost:8080' target="_black">
    http://localhost:8080
    <ExternalLinkIcon/>
</a>  and wait for application up.
</p>

<p>
    3. Go to Drill Admin <a href='http://localhost:8091' target="_black">
    http://localhost:8091
    <ExternalLinkIcon/>
</a>  and register agents.
</p>

<AdditionalInformation className="space-y-2">

Don't forget to set target host "http://localhost:8080/" for frontend agent.

</AdditionalInformation>

4. Run tests:
```java
./gradlew clean :build1:test
```
See results in the Drill Admin and finish scopes.

5. Deploy the second build of the app:
```java
./demo-deploy-build2.sh
```
See new build notifications and diffs in the Drill Admin.

6. Run tests:
```java
./gradlew clean :build2:test
```
See results in the Drill Admin and finish scopes.

7. Deploy the third build of the app:
```java
./demo-deploy-build3.sh
```
See new build notifications and diffs in the Drill Admin.

8. Run tests:
```java
./gradlew clean :build3:test
```
See results in the Drill Admin and finish scopes.

9. Use browser extension for manual testing:
<Link to='/docs/installation/extension' target="_black">
    Chrome Browser Extension Installation
    <ExternalLinkIcon/>
</Link>

Also, you can down demo setup:
```java
./demo-down.sh
```

</ExpandCollapse>

<!-- JS BLOCK -->

<ExpandCollapse Icon={<JsLogo />} title="JS App">

**Clone the example repository:**

```java
git clone https://github.com/Drill4J/todomvc
```

**Run Drill4J services**:

1. Run:
```java
docker-compose -f docker-compose.services.yml up -d
```

2. Run `docker ps` and check that all services have started: `drill-admin`, `admin-ui`, `jsagent` and  `mongo`.

<p>
    3. Open Drill Admin UI at <a href='http://localhost:9091' target="_black">
    http://localhost:9091
    <ExternalLinkIcon/>
</a>
</p>

**Run sample application**:
1. Run:

```java
docker-compose up -d
```
<p>
    2. Wait for "todomvc-typescript-angular" agent to appear in Admin UI at <a href='http://localhost:9091' target="_black">
    http://localhost:9091
    <ExternalLinkIcon/>
</a>
</p>

3. Press **Register** button.

4. Click through *Step 1* and *Step 2*, but **make sure to enable** Test2Code plugin at *Step 3*.

<p>
    5. Open sample application at <a href='http://localhost:8080' target="_black">
    http://localhost:8080
    <ExternalLinkIcon/>
</a>
</p>

6. Use browser extension for manual testing:
<Link to='/docs/installation/extension' target="_black">
    Chrome Browser Extension Installation
    <ExternalLinkIcon/>
</Link>

</ExpandCollapse>
